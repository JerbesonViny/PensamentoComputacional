## Estrutura de repetição

<p>
Escreva um programa que lê a quantidade dos alunos de uma turma. Em seguida, o programa deve ler os nomes de cada aluno e imprimí-los na ordem inversa.
</p>

### Exemplo

Entrada:

```
Quantos nomes? 3
João
Paulo
Augusto
```

Saída:

```
Você digitou:
Augusto
Paulo
João
```
