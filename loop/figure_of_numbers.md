## Estrutura de repetição

<p style="text-align: justify">
Crie um programa que leia um número N (0<N<10) e que imprima uma sequência de números conforme mostrado no exemplo:
</p>

### Exemplos

Primeira entrada:

```
n: 5
```

Saída:

```
1

22

333

4444

55555
```

Segunda entrada:

```
n: 9
```

Saída:

```
1

22

333

4444

55555

666666

7777777

88888888

999999999
```
